# Super Platformer Adventure

A browser-based side-scrolling platformer inspired by classic 8-bit games.

## Features
- 3 complete levels with increasing enemy difficulty
- Collectible coins and breakable bonus blocks
- Enemy stomp combat and life system
- Level progression, score tracking, and win/game-over flow
- Camera-follow side scrolling and simple HUD

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`.

## Controls
- `←` / `→`: move
- `Space` or `↑`: jump
- `Shift`: run faster
